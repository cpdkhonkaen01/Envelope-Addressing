# ระบบจ่าหน้าซองจดหมายราชการ

ระบบค้นหา เพิ่ม และพิมพ์จ่าหน้าซอง โดยใช้ Google Sheets เป็นฐานข้อมูล พร้อมตัวเลือกเปิด–ปิดตราครุฑเหนือชื่อหน่วยงานผู้ส่งในหน้าพิมพ์

```text
หน้าเว็บ Vercel ↔ Google Apps Script ↔ Google Sheets
```

หน้าเว็บเชื่อมฐานข้อมูลอัตโนมัติ ไม่มีช่องตั้งค่า Apps Script ให้ผู้ใช้เห็น และแบ่งรายชื่อเป็น 4 ประเภท

- หน่วยงานราชการ
- สหกรณ์และกลุ่มเกษตรกร
- ภาคเอกชน
- บุคคล

## 1. เตรียม Google Sheets

สร้างชีตชื่อ `Recipients` และใช้หัวตารางดังนี้

```text
id,category,prefix,firstName,lastName,position,department,address1,subdistrict,district,province,postalCode,active,createdAt
```

สามารถนำเข้า `sample-data/recipients.csv` หรือเรียกฟังก์ชัน `setupRecipientsSheet()` ใน Apps Script เพื่อสร้างหัวตารางได้

## 2. ตั้งค่า Google Apps Script

1. เปิด Google Sheet แล้วเลือก **Extensions > Apps Script**
2. นำโค้ดจาก `apps-script/Code.gs` ไปแทนโค้ดเดิม
3. แก้ `SPREADSHEET_ID` ให้เป็นรหัสจาก URL ของ Google Sheet
4. ที่เมนู **Services (บริการ)** กด **+ Add a service (เพิ่มบริการ)**
5. เลือก **Google Sheets API**, เวอร์ชัน **v4**, ตัวระบุ **Sheets** แล้วกด **บันทึก**
6. เปิด **Project Settings > Script Properties**
7. เพิ่ม Property ชื่อ `WRITE_KEY` และกำหนดรหัสผู้ดูแลที่คาดเดายาก
8. เลือก **Deploy > New deployment > Web app**
9. ตั้ง **Execute as: Me** และ **Who has access: Anyone**
10. คัดลอก URL ที่ลงท้ายด้วย `/exec`

การเพิ่มบริการ Google Sheets API ใน Apps Script วิธีนี้ไม่ต้องนำ API key ไปใส่ในหน้าเว็บ โดย `Code.gs` จะอ่านรายชื่อผ่านบริการ `Sheets` และเก็บผลไว้ชั่วคราว 2 นาทีเพื่อให้โหลดซ้ำเร็วขึ้น

เมื่อแก้ `Code.gs` ต้องสร้าง Version ใหม่ใน **Manage deployments** เพื่อให้ Web app ใช้โค้ดล่าสุด

## 3. เชื่อมหน้าเว็บกับ Apps Script

เปิด `config.js` แล้วใส่ URL `/exec`

```js
window.ENVELOPE_APP_CONFIG = {
  appsScriptUrl: "https://script.google.com/macros/s/DEPLOYMENT_ID/exec",
  sender: "สำนักงานสหกรณ์จังหวัดขอนแก่น",
  senderAddress: "ที่อยู่หน่วยงานผู้ส่ง",
  paperSize: "DL",
};
```

จากนั้นรันคำสั่งนี้เพื่อสร้าง `index.html` แบบไฟล์เดียว

```text
node scripts/build-standalone.mjs
```

รูปตราครุฑโหลดจาก URL ที่กำหนดใน `garudaImage` ภายใน `config.js` จึงไม่ทำให้ไฟล์ `index.html` มีขนาดใหญ่ และไม่ต้องอัปโหลดรูปแยกขึ้น Vercel

หากแก้ `index.html` โดยตรง ให้ค้นหา `appsScriptUrl` และใส่ URL ในส่วน `app-config`

## 4. เพิ่มข้อมูลจากหน้าเว็บ

กด **เพิ่มรายชื่อผู้รับ** เลือกประเภท กรอกข้อมูล และใส่รหัสผู้ดูแลเดียวกับ `WRITE_KEY` ข้อมูลจะบันทึกลงชีต `Recipients` โดยตรง ประเภทบุคคลไม่บังคับชื่อหน่วยงาน แต่ต้องระบุชื่อหรือนามสกุล รหัสผู้ดูแลจะเก็บเฉพาะใน Session ของเบราว์เซอร์และจะหายเมื่อปิดเบราว์เซอร์

รายการที่บันทึกแล้วจะมีปุ่ม **แก้ไข** ในตารางรายชื่อ เมื่อบันทึกการแก้ไข ระบบจะค้นหาจาก `id` และอัปเดตแถวเดิมในชีต `Recipients` โดยไม่สร้างรายการซ้ำ ความสามารถนี้ต้องใช้ `apps-script/Code.gs` รุ่นล่าสุดและสร้าง Version ใหม่ใน **Manage deployments**

ช่องรหัสไปรษณีย์รองรับตัวเลข 5 หลักทั้งเลขอารบิก เช่น `40000` และเลขไทย เช่น `๔๐๐๐๐` โดยระบบจะเก็บและพิมพ์ตามรูปแบบที่กรอก

หน้าพิมพ์จะแสดงชื่อเขตการปกครองแบบเต็มโดยไม่เว้นช่องว่างหลังคำนำหน้า เช่น `ตำบลบ้านค้อ อำเภอเมืองขอนแก่น` และ `จังหวัดขอนแก่น` ส่วนรหัสไปรษณีย์จะแสดงแยกเป็นบรรทัดสุดท้าย

ไม่ควรเขียน `WRITE_KEY` ลงใน `index.html`, `config.js` หรือ GitHub เพราะ Repository และหน้าเว็บอาจเปิดเป็นสาธารณะ

## ตั้งค่าหน่วยงานผู้ส่งและรูปแบบการพิมพ์

กด **แก้ไขข้อมูลผู้ส่งและตำแหน่ง** ในแผงพร้อมพิมพ์ เพื่อแก้ไขชื่อหน่วยงาน ที่อยู่ผู้ส่ง เลขที่หนังสือ ตำแหน่งตราครุฑ ขนาดตราครุฑ ขนาดฟอนต์ข้อมูลผู้ส่ง 7–20 พอยต์ และระยะจากขอบกระดาษ กรณีวางตราครุฑด้านซ้าย สามารถใช้ช่อง **เลื่อนข้อความลง (มม.)** เพื่อจัดชื่อหน่วยงานให้ตรงกับระดับเท้าครุฑได้ ค่าทั้งหมดเก็บไว้ในเบราว์เซอร์ของเครื่องที่ตั้งค่า

กด **แก้ไขกรอบค่าฝากส่ง** เพื่อแก้ข้อความเลขที่ใบอนุญาตและชื่อที่ทำการไปรษณีย์ กรอบมีขนาดคงที่ 5 × 2.5 เซนติเมตร และสามารถปรับระยะจากขอบบน ขอบขวา และขนาดตัวอักษรได้

ตราครุฑใช้ความสูงมาตรฐาน 1.5 เซนติเมตร (15 มิลลิเมตร) สำหรับซอง DL ซอง C5 กระดาษ A4 แนวตั้ง และกระดาษ A4 แนวนอน โดยยังสามารถปรับขนาดเองได้จากหน้าตั้งค่าผู้ส่ง

ขนาดฟอนต์ข้อมูลผู้รับปรับได้จากแผง **พร้อมพิมพ์** ช่วง 8–24 พอยต์ และมีผลกับภาพตัวอย่างและหน้าพิมพ์ทุกขนาด

กด **ปรับตำแหน่งและระยะผู้รับ** เพื่อเลื่อนข้อความผู้รับขึ้น–ลง ซ้าย–ขวา และปรับระยะห่างระหว่างบรรทัด ค่าเหล่านี้ใช้กับทั้งภาพตัวอย่างและหน้าพิมพ์ และบันทึกไว้ในเบราว์เซอร์ของเครื่องที่ตั้งค่า

แผง **พร้อมพิมพ์** แสดงเป็นแนวนอนเหนือรายชื่อ เพื่อให้ตารางใช้ความกว้างเต็มหน้า และคอลัมน์ที่อยู่จะแสดงข้อมูลผู้รับครบตั้งแต่เลขที่ หมู่ ถนน ตำบล อำเภอ จังหวัด และรหัสไปรษณีย์

ตัวเลือก **แสดงข้อมูลผู้ส่ง** ใช้เปิด–ปิดชื่อหน่วยงานผู้ส่ง ที่อยู่ เลขที่หนังสือ และตราครุฑทั้งหมด เหมาะสำหรับซองที่ใช้ตราประทับผู้ส่งแทนการพิมพ์

## 5. เผยแพร่ผ่าน GitHub และ Vercel

สำหรับหน้าเว็บที่พร้อมใช้งาน ให้อัปโหลดไฟล์ต่อไปนี้ขึ้น GitHub

- `index.html`
- `vercel.json`
- `.vercelignore`

จากนั้น Import Repository เข้า Vercel โดยเลือก Framework Preset เป็น **Other** และไม่ต้องกำหนด Build Command เมื่อ Push หรือแก้ไฟล์ใน branch `main` Vercel จะเผยแพร่รุ่นใหม่อัตโนมัติ

## การแก้ปัญหา

- สถานะขึ้นว่า **ยังไม่ได้ตั้งค่า Apps Script**: ตรวจ `appsScriptUrl` ใน `index.html`
- อ่านข้อมูลได้แต่เพิ่มไม่ได้: ตรวจ `WRITE_KEY` และ Deploy Apps Script เป็น Version ล่าสุด
- รายชื่อเดิมที่ระบุประเภท “สหกรณ์” หรือ “กลุ่มเกษตรกร” จะถูกรวมเป็น “สหกรณ์และกลุ่มเกษตรกร” อัตโนมัติ
- รายชื่อเดิมที่ไม่มีประเภท: ระบบจะจัดชื่อที่มีคำว่า “สหกรณ์” หรือ “กลุ่มเกษตรกร” เข้าหมวดรวม และตรวจชื่อบริษัทเบื้องต้นเป็น “ภาคเอกชน” ส่วนรายการอื่นจะอยู่ใน “หน่วยงานราชการ”
